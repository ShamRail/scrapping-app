Тема проекта: Функционирование киберфизических систем и их разработка

Данное веб-приложение представляет собой интерфейс пользователя для обращения к БД со статьями и публикациями.

Источники данных:
  • Источник публицаций: Google Scholar
  • Источник статей: Поисковый движок Google
Примечание: Первый поддерживает только английский язык поиска. Второй как английский так и русский языки.

Используемые технологии:
Технология Применение
Java и фреймворк Реализация основной фунциональности приложения
Python и библиотека Извлечение данных из Google Scholar
Движок СУБД: PostgreSQL Хранение данных
Amazon Web Services Развертывание БД и веб-приложения
Google Custom Search Поиск статей посредством движка Google
Bootstrap Стилизация веб-приложения
